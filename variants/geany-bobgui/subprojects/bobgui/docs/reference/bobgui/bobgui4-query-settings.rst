===================
bobgui4-query-settings
===================

------------------------------------
Print name and value of BOBGUI settings
------------------------------------

:Version: BOBGUI
:Manual section: 1
:Manual group: BOBGUI commands

SYNOPSIS
--------

|   **bobgui4-query-settings** [PATTERN]

DESCRIPTION
-----------

``bobgui4-query-settings`` prints both name and value of all properties available
in the ``BobguiSettings`` class. Optionally, you can filter which properties to
list by specifying a ``PATTERN``.
