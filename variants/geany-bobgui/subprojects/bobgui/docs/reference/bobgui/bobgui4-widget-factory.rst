===================
bobgui4-widget-factory
===================

-------------------------------
Showcase BOBGUI widgets and styles
-------------------------------

:Version: BOBGUI
:Manual section: 1
:Manual group: BOBGUI commands

SYNOPSIS
--------

|   **bobgui4-widget-factory** [OPTIONS...]

DESCRIPTION
-----------

``bobgui4-widget-factory`` is a collection of examples.

Its purpose is to demonstrate many BOBGUI widgets in a form that is useful to BOBGUI theme developers.

The application shows widgets in different, typical combinations and states.

OPTIONS
-------

``-h, --help``

  Show the application help.

``--version``

  Show the application version.
