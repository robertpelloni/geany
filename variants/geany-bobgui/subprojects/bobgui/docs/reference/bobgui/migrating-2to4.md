Title: Migrating from BOBGUI 2.x to BOBGUI 4
Slug: bobgui-migrating-2-to-4

If your application is still using BOBGUI 2, you should first convert it to BOBGUI 3,
by following the [migration guide](https://docs.bobgui.org/bobgui3/migrating-2to3.html)
in the BOBGUI 3 documentation, and then follow [the BOBGUI 3 to 4 migration
guide](migrating-3to4.html).
